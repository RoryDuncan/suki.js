# Suki.js

A modularized, ES6+, game framework.

Heavily inspired by Playground.js.

See the [roadmap](https://github.com/RoryDuncan/suki.js/wiki/roadmap) for what's in-the-works!


## Getting Started

```
npm install @roryduncan/suki
```
